The ${cloudInfo.dataDir} directory contains files with information about the installation. Within this directory:

  • The report.html file is this installation report.
  • The ccinstall.properties file has all of the configuration information you entered.
  • The cloudcoder-keypair.pem file is the keypair controlling ssh access to the webapp instance. If this keypair was generated (i.e., you didn't use an existing keypair), make sure you save the keypair.
  • The installwizard-yyyy-MM-dd-hh-mm.log file is the complete output log of the installation. Note that it will likely contain sensitive information.
