This page allows you to specify dynamic DNS provider account information. If you choose to use dynamic DNS and enter your account information, the installation wizard will be able to create a DNS entry automatically, and also issue and install an SSL certificate using Let's Encrypt. (Note that if you don't use dynamic DNS, the wizard can issue and install an SSL certificate later, once you've created a DNS entry.)

Currently, Duck DNS is the only supported dynamic DNS provider.

If you choose to use dynamic DNS, make sure that you have already created the hostname you will be using.
