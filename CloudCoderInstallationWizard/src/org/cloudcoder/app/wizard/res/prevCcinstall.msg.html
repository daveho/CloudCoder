It looks like you have run the CloudCoder installation wizard previously. Check the box labeled Load previous values if you would like to load the previous values.

This is a very good idea if

  1. You did a dry run previously, and now you are ready to install for real, or
  2. You are re-running the installation wizard to issue and install a Let's Encrypt SSL certificate

Loading the previous configuration values means you won't have to re-enter them. You will still be able to make changes if you desire.
